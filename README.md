# Onmeds

Online Portal for selling COVID-19 Test Packages to User.

Feauters:
Logging User
Homepage
Cart

# Tech
Reactjs,
Nodejs,
Redis,
MySQL

### Installation

```sh
$ cd netmeds-back-end
$ npm install 
$ node src/app.js
```
