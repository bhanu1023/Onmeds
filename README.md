# Onmeds

Online Portal for selling COVID-19 Test Packages to User.

Feauters:
Logging User
Homepage
Cart

# Tech
Reactjs,
Nodejs,
Redis,
MySQL

### Requirement
MySQL server with a user with username and hashed password stored <br/>
Redis Server 

### Installation

```sh
$ cd netmeds-back-end
$ npm install 
$ node src/app.js
```

### Pages

If server is running at localhost:3001 then <br/>
Homepage: localhost:3001
